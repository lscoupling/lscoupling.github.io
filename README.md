# 🌐 個人網站介紹

這是一個用來展示我個人創作、專案整理與內容規劃的網站，定位為長期維護的個人入口頁。  
整體以前端效能、可讀性與維護性為核心，使用現代前端工具鏈打造，並以靜態網站形式部署在 GitHub Pages 上。

網站內容包含個人介紹、專案展示與規劃頁面，適合作為作品集、技術紀錄與對外分享的平台。

## 📍 網站連結
https://lscoupling.github.io/#/

## 🛠 技術架構

- React：負責整體 UI 組件化與頁面邏輯
- Vite：提供快速的開發體驗與高效的建置流程
- GitHub Pages：作為靜態網站部署與對外公開平台

## ⚙️ 本地開發與部署

### 安裝依賴
```bash
npm install
```

### 本地開發（啟動 dev server）
```bash
npm run dev
```

### 建立正式版（產生 dist/）
```bash
npm run build
```

### 部署到 GitHub Pages
```bash
npm run deploy
